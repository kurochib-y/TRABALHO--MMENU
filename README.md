Um menu de opções é uma interface que apresenta ao usuário uma lista de ações disponíveis dentro de um sistema ou programa. Cada opção possui uma função específica, como cadastrar informações, consultar dados, realizar alterações, excluir registros ou encerrar o sistema. O objetivo do menu é facilitar a navegação e tornar o uso do programa mais organizado, permitindo que o usuário escolha rapidamente a operação desejada por meio de números, letras ou comandos. Dessa forma, o menu contribui para uma experiência mais simples, prática e eficiente. - Lau e Mari
